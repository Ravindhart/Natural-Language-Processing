1. Please place jars for the following libraries into the program class path, you can use the downlod url mentioned in the report:

Stanford CoreNLP,
Stanford Named Entity Recognizer (NER)
Stanford Log-linear Part-Of-Speech Tagger
MIT Java Wordnet Interface – Princeton Wordnet

2. Compile the programs using the commands
javac StringConstants.java  
javac StanfordLemmatizer.java  
javac Stemmer.java          
javac StanfordNER.java
javac TermCountWeight.java     
javac WordCountMap.java
javac FileReader.java  
javac TopicDetectionUtil.java
javac NLP.java

3. Run the program using the commad:
java NLP
